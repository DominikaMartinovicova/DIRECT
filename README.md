# DIRECT
Analysis of spatial single cell Xenium data from patients in DIRECT cohort.

## Repository Structure

- `hpc_scripts/` - Backup of scripts from HPC cluster for analysis workflows

## HPC Scripts Backup

This repository serves as a backup and version control for scripts used on the HPC cluster. See `hpc_scripts/README.md` for instructions on syncing scripts between this repository and the HPC cluster.
